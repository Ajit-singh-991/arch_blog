# Contributing to Architectural World

Thank you for your interest in contributing to Architectural World! We welcome contributions from everyone.

## Code of Conduct

By participating in this project, you are expected to uphold a safe, inclusive, and welcoming environment for everyone.

## How Can I Contribute?

*   **Reporting Bugs:** Open an issue describing the bug and how to reproduce it.
*   **Suggesting Enhancements:** Open an issue with your proposed features or improvements.
*   **Pull Requests:**
    1.  Fork the repo and create your branch from `main`.
    2.  If you've added code that should be tested, add tests.
    3.  Ensure the test suite passes.
    4.  Make sure your code follows the existing style (Future/Glassmorphism theme).
    5.  Issue that pull request!

## Style Guide

*   **CSS:** Use CSS variables defined in `:root` for colors (e.g., `--primary-neon`).
*   **Components:** Angular Standalone components are preferred.
*   **Images:** Use high-quality architectural images (Unsplash preferred).

Thank you for contributing!
